---

Tokens are digital assets that represent ownership over diverse categories of assets. Tokenization
enables the digitalization of property rights. Tokens on Trezoa are referred to as TPL
([Trezoa Program Library](https://github.com/trezoa-program)) Tokens.

The following guides demonstrate the most common tasks a Trezoa developer should be familiar with if
building an application that utilizes tokens.
